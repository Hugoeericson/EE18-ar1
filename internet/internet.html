Frågor om Internet

Vad är En Standard? En standard är något som alla har tillgång till, tillexempel ett eluttag i en väg.
Vad har hänt med HTML sen det skapades? Sen HTML skapades 1992 så har det hänt mycket! På 22 år har det redan kommit ut 5 nya verisioner av html.
Vad är det fördelar och nackdelar med webben? Fördelar med webben är att den skappar många möjligheter som jobb. Nackdelar är att Man köpa dråger, bli hackad osv.

Var kommer HTML från?

HTML en förkortning för HyperText Markup Language. Är ett prorammerings språk som används när man skapa en egen webbsida från grunden. Det var Tim Berner-Lee som hittade på själva HTML spårket.

Vad innebär webben i vårt samhälle och dagliga liv?

År 2019 använder vi webben till nästan allt. Vi jobbar, köper kläder, chattar, gör skol arbete osv. Så webben betyder alltså väldigt mycket för oss. Våra liv lustar nästan på webben.

Hur fugerar webben?

Webben fungerar med hjälp av serverar som håller sidan uppe och nätverk för att sidan ska kunna delas med andra. Ett exepel på hur man kan förklara hur webben fungerar är att sidan är som ett paket. Om man ska till exepel söka på afftonbladet så skickar du en förfrågon till afftonbladet serverar om du får gå på sidna. Om du nu får komma in så skickas ett slags paket från deras server address till din framdörr. Detta händer dock på någon milesekund.
